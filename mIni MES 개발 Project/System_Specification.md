
# 프로젝트명: mini MES 개발 Project

## 1. 프로젝트 개요 
- 해당 프로젝트는 라면제조 공정 MES 도입 위한 개발 프로젝트 입니다.
- 기안자: 이민하
  
---

## 2. 개발 목적 
- MES 도입으로 공정관리 전산화 및 자동화
- 자동화 달성으로 불필요한 Cost CI(Cost Inovation)

## 3. 대상 사용자
- 생산관리 실장 → 생산실적 관리
- 공정 관리자 및 부관리자 → 세부공정 관리
- 자재 담당자 → 사내 재고 관리
- 품질 담당자 → 품질검사 관리
- 영업 담당자 → 출하 관리

## 4. 주요기능 
### 1)재고관리
- 협력사 자재 입고 등록
- 생산후 잔여 재고 등록
### 2)작업지시
- 생산계획분 작업지시
### 3)생산실적
- 생산실적 조회
- 생산완료 등록
### 4)품질검사
- IQC(input quality control) 검사 판정
- LQC(line quality control) 불량품 및 불량원인(카테고리화) 등록  
- OQC(output quality control) 검사 판정 
### 5)출하관리
- 출하 등록
- 출하 이력 조회

## 5. 개발환경
### 1) 하드웨어/ 설비 
- 기능: 실시간 데이터 수집 및 처리 엔진
- 개발언어: C/C++, RUST
### 2) Data Base 
- 기능: Raw 데이터 저장 
- 개발언어: SQL(SQLITE)
### 3) 프론트 엔드 
- 기능: 데이터 조회 및 등록
- 개발언어: Python(streamlit)
